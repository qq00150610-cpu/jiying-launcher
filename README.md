# 极影桌面 jiying

极影桌面是一款融合布丁UI与氢桌面优势的车载智能桌面应用，支持Android 6.0+系统。

## 功能特点

### 主界面
- 大字体、大图标设计，适合车机横屏使用
- 可隐藏的快捷插件区域（导航+音乐控制）
- 支持动态壁纸/静态壁纸
- 应用图标网格布局

### 智能助理
- 左上角麦克风入口
- AI对话界面
- 语音交互功能

### 快捷插件
- 导航插件（支持调起高德等导航应用）
- 音乐插件（支持控制音乐播放）
- 可隐藏/显示

### 壁纸中心
- 壁纸分类展示
- 壁纸预览和设置
- 支持动态壁纸

### 轻应用中心
- 轻应用列表展示
- 分类：影音、播客、游戏、资讯、学习
- 快速启动

### 控制中心（下拉式）
- 下拉手势触发
- 昼夜模式切换
- 屏幕亮度调节
- 媒体/系统音量调节
- 网络/蓝牙开关

### 车主服务
- 文件管理（读取车机本地文件）
- 应用备份与恢复
- 远程传输（扫码上传文件）
- 垃圾清理
- 流量监控

## 多布局模式系统

极影桌面支持6种首页布局模式，可在长按导航按钮进入布局模式选择器：

### 1. 默认模式
经典双卡片布局，地图与音乐并列显示，平衡实用。

### 2. 混合模式
紧凑混合布局，地图小窗+快捷应用栏，适合大多数车机屏幕。

### 3. 地图优先模式
地图占据主要视野，驾驶导航更清晰，适合导航为主的使用场景。

### 4. 音乐优先模式
音乐界面最大化，适合音乐爱好者，大面积展示播放信息。

### 5. 极简模式
极简设计，只保留最常用的功能入口，适合追求简洁的用户。

### 6. CarPlay风格模式
类似CarPlay的大图标风格，简洁直观，图标尺寸更大。

## 多设备适配系统

### 支持的设备类型
| 设备类型 | 推荐分辨率 | 适配布局 |
|---------|-----------|---------|
| 7寸车机 | 1024x600 | 混合模式 |
| 10寸车机 | 1280x800 | 地图优先 |
| 12寸大屏车机 | 1920x1200 | CarPlay风格 |
| 特斯拉风格大屏 | 1920x1200+ | 极简模式 |
| 竖屏车机 | 600x1024 | 默认模式 |
| 手机兼容 | 360x640+ | 默认模式 |

### 屏幕适配功能
- 自动检测当前设备类型
- 一键应用设备预设配置
- 自定义布局参数调整
- 横屏/竖屏自动切换

### 布局模式入口
- **长按导航按钮**：打开布局模式选择器
- **长按首页按钮**：打开设备配置界面

## 悬浮地图功能

借鉴氢桌面的悬浮地图核心功能：
- 支持高德地图、百度地图悬浮显示
- 画中画模式（Picture-in-Picture）
- 可拖拽移动位置
- 支持折叠/展开切换
- 四种尺寸模式：迷你、小、中、大

## 计划任务功能

借鉴氢桌面的计划任务功能：
- 定时启动指定应用（如开机自动启动地图）
- 延迟返回桌面
- 支持单次/每天/每周等多种重复模式
- 驾驶场景自动化

## 小白点快速返回

- 悬浮球快速操作
- 支持返回桌面、最近任务、锁屏等快捷操作
- 可自定义快捷功能

## 双主题系统

### 布丁UI风格（默认）
- 现代化卡片设计
- 活泼配色
- 圆角适中
- 功能丰富的快捷入口

### 氢桌面风格
- 简洁卡片化设计
- 大面积留白

- 清爽的配色

## 技术栈

- **语言**: Kotlin
- **最低SDK**: Android 6.0 (API 23)
- **目标SDK**: Android 14 (API 34)
- **架构**: 单Activity + 多Fragment
- **UI组件**: Material Design 3
- **图片加载**: Glide

## 核心工具类说明

### ScreenAdapter.kt - 屏幕适配工具
- 获取屏幕尺寸、分辨率、密度
- 判断设备类型（手机、平板、车机）
- dp/px/sp单位转换
- 状态栏/导航栏高度获取

### LayoutModeManager.kt - 布局模式管理
- 管理6种布局模式
- 布局参数配置（地图比例、音乐比例、Dock高度）
- 布局切换和保存

### MultiDeviceConfig.kt - 多设备配置
- 预设多种设备配置（7寸、10寸、12寸车机）
- 自动检测设备类型
- 一键应用设备预设

### ScheduledTaskManager.kt - 计划任务管理
- 创建定时启动应用任务
- 延迟返回桌面任务
- 支持多种重复模式
