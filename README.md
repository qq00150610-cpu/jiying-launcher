# 极影桌面 jiying

## 功能特点

### 主界面
- 大字体、大图标设计，适合车机横屏使用
- 可隐藏的快捷插件区域（导航+音乐控制）
- 支持动态壁纸/静态壁纸
- 应用图标网格布局

### 智能助理
- 左上角麦克风入口
- AI对话界面
- 语音交互功能

### 快捷插件
- 导航插件（支持调起高德等导航应用）
- 音乐插件（支持控制音乐播放）
- 可隐藏/显示

### 壁纸中心
- 壁纸分类展示
- 壁纸预览和设置
- 支持动态壁纸

### 轻应用中心
- 轻应用列表展示
- 分类：影音、播客、游戏、资讯、学习
- 快速启动

### 控制中心（下拉式）
- 下拉手势触发
- 昼夜模式切换
- 屏幕亮度调节
- 媒体/系统音量调节
- 网络/蓝牙开关

### 车主服务
- 文件管理（读取车机本地文件）
- 应用备份与恢复
- 远程传输（扫码上传文件）
- 垃圾清理
- 流量监控

## 双主题系统

### 布丁UI风格（默认）
- 现代化卡片设计
- 活泼配色
- 圆角适中
- 功能丰富的快捷入口

### 氢桌面风格
- 简洁卡片化设计
- 大面积留白
- 圆角卡片
- 清爽的配色

## 技术栈

- **语言**: Kotlin
- **最低SDK**: Android 8.0 (API 26)
- **目标SDK**: Android 14 (API 34)
- **架构**: 单Activity + 多Fragment
- **UI组件**: Material Design 3
- **图片加载**: Glide

## 注意事项

- 这是一个Android车机桌面应用，需要考虑驾驶安全
- 界面元素要足够大，便于驾驶时操作
- 支持语音控制相关功能
- 考虑权限申请（文件访问、悬浮窗等）
- 保持代码结构清晰，便于后续二次开发
