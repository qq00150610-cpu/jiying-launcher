# 极影桌面开发说明

## 项目概述

极影桌面（JiYing Launcher）是一款基于Android的车机桌面应用，复刻自布丁UI风格，并支持氢桌面风格切换。

## 技术实现

### 包名
- `com.jiying.launcher`

### 签名配置
- keyAlias: `jiying`
- storePassword: `qq900236`
- keyPassword: `qq900236`

### 最低要求
- Android 8.0 (API 26)
- 目标SDK: Android 14 (API 34)

## 主题系统

### 布丁UI风格
```kotlin
ThemeManager.THEME_BUDING
- 主色调: #2196F3 (蓝色)
- 强调色: #FF5722 (橙红色)
- 卡片圆角: 16dp
- 卡片阴影: 8dp
```

### 氢桌面风格
```kotlin
ThemeManager.THEME_HYDROGEN
- 主色调: #4CAF50 (绿色)
- 强调色: #03A9F4 (浅蓝色)
- 卡片圆角: 24dp
- 卡片阴影: 4dp
```

### 切换主题
```kotlin
// 切换主题
ThemeManager.toggleTheme()

// 获取当前主题
ThemeManager.getCurrentTheme()

// 设置主题
ThemeManager.setTheme(ThemeManager.THEME_BUDING)  // 或 THEME_HYDROGEN
```

## 功能模块

### 1. 主界面 (MainActivity)
- 壁纸背景显示
- 时间日期显示
- 快捷插件（导航+音乐）
- 应用网格列表
- 底部导航栏
- 控制中心

### 2. 智能助理 (AssistantActivity)
- 语音输入入口
- AI对话界面（待实现）

### 3. 壁纸中心 (WallpaperPreviewActivity)
- 壁纸预览
- 壁纸设置

### 4. 轻应用中心 (AppsCenterActivity)
- 分类应用列表
- 应用快速启动

### 5. 控制中心 (ControlCenterActivity)
- 亮度调节
- 音量调节
- WiFi开关
- 蓝牙开关
- 夜间模式

### 6. 车主服务 (FileManagerActivity)
- 文件管理
- 应用备份
- 远程传输
- 垃圾清理
- 流量监控

## 服务层

### DesktopService
桌面服务，处理后台任务。

### WallpaperService
动态壁纸服务。

### BootCompletedReceiver
开机广播接收器，自动启动桌面。

## 构建说明

项目使用GitHub Actions进行构建：

1. 配置Secrets:
   - `KEYSTORE_BASE64`: keystore文件Base64编码
   - `KEYSTORE_PASSWORD`: qq900236
   - `KEY_PASSWORD`: qq900236

2. 推送代码到GitHub自动触发构建

3. 构建产物在Actions artifacts中下载
